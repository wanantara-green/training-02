# Pelatihan Perencanaan Zonasi Spasial Kabupaten Luwu

> Training online ini dipublikasikan di **[latih-02.wanantara.org](https://latih-02.wanantara.org)**

Website pelatihan penguatan kapasitas **Perencanaan Zonasi Spasial** untuk Kabupaten Luwu, Sulawesi Selatan.

## Halaman

| Halaman | Deskripsi |
|---|---|
| `index.html` | Beranda — pengantar pelatihan, alur belajar, sasaran peserta, dan section Tentang |
| `modul.html` | Pusat pembelajaran — 6 modul dengan materi, kuis interaktif, dan progres tracking |
| `map.html` | WebGIS interaktif — 30 layer GeoJSON tematik, panel layer, legenda, info atribut, pengukuran jarak, dan download |

## Modul Pelatihan

1. **Konsep Dasar Perencanaan Zonasi Spasial** — Fondasi: definisi, urgensi, dasar hukum
2. **Metodologi Analisis Spasial (MCE-GIS)** — Standardisasi, pembobotan, agregasi, hukum minimum
3. **Enam Kelompok Indikator Spasial (X1–X6)** — 29 indikator, bobot, kerangka IPCC AR5
4. **Membaca & Menggunakan Peta Zonasi** — Simbologi, legenda, klasifikasi 5 kelas, interpretasi
5. **Penerapan Zonasi dalam Peraturan & Perizinan** — KKPR, revisi RTRW, ketentuan kegiatan
6. **Pemantauan & Evaluasi Zonasi** — M&E, InaRISK/BPBD, siklus pemutakhiran

## Tech Stack

HTML5, Tailwind CSS (CDN), MapLibre GL JS 3.6, Turf.js 6, Vanilla JavaScript. Static site — tanpa build step, tanpa backend. Hosting di GitHub Pages (`training-02.wanantara.org`).

### PIN Reset Progres

Untuk mereset seluruh progres belajar, klik tombol **Reset Progres** (merah) pada halaman modul, lalu masukkan PIN:

| Fungsi | PIN |
|---|---|
| Reset Progres Belajar | **1234** |

Fitur reset memungkinkan peserta lain memulai pelatihan dari awal pada perangkat yang sama.

### Cara Penggunaan

1. **Membuka kuis**: Buka modul yang diinginkan, scroll ke section **Kuis**, masukkan PIN modul terkait, klik **Buka Kuis**.
2. **Reset progres**: Klik tombol **Reset Progres** di panel Progres Belajar, masukkan `1234`, klik **Reset**.
3. PIN bersifat rahasia untuk fasilitator dan hanya digunakan saat sesi pelatihan.
